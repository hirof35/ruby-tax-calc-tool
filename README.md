# 税務・書類計算マルチツール (Tax & Invoice Calculator)

Ruby/Tk（GUI）で動作する、個人事業主・フリーランス・経理担当者向けの簡易デスクトップ計算ツールです。
タブを切り替えることで、実務で頻出する「源泉徴収・手取り」および「消費税の内税・外税」を瞬時に計算できます。

## 💡 主な機能

### 1. 源泉徴収・手取り計算タブ
- **2段階スライド税率対応**: 報酬額が100万円を超える場合の自動増税（10.21% → 20.42%）に完全対応しています。
- **消費税別給対応**: 請求書に消費税を別途記載して請求するパターンの計算（消費税10%のオン／オフ）がチェックボックスで切り替え可能です。

### 2. 消費税・内外税一発計算タブ
- **内外税の相互逆算**: 外税（税別入力から税込を算出）と内税（税込入力から本体価格と税額を算出）をラジオボタンで切り替えられます。
- **インボイス対応（税率選択）**: スライダーを動かすことで、軽減税率（8%）と標準税率（10%）の計算を切り替えられます。
- **端数処理**: 実務で一般的な「消費税端数の切り捨て」ロジックを組み込んでいます。

## 🛠️ 動作環境・前提条件

- **Ruby**: 3.0 以降（推奨）
- **OS**: Windows / macOS / Linux
- **必須Gem**: `tk`

> ⚠️ **注意**: Ruby 3.0 以降では、Tkライブラリが標準添付されていません。実行前に以下の環境構築を行ってください。

### 1. ターミナルでの事前準備（環境に応じたTkのインストール）

#### Windows (RubyInstaller)
RubyInstallerのDevkitを利用して、コマンドプロンプトまたはPowerShellで以下を実行します。
```bash
gem install tk
macOS (Homebrew使用)
先にシステム側へ tcl-tk を導入する必要があります。

Bash
brew install tcl-tk
gem install tk -- --with-tk-dir=$(brew --prefix tcl-tk)
2. 起動方法
リポジトリ（またはフォルダ）に移動し、以下のコマンドを実行します。

Bash
ruby calc-tool.rb
